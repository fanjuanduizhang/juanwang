# SurveyKing 部署指南

## 📋 环境要求

| 软件 | 版本要求 | 说明 |
|------|---------|------|
| JDK | 17+ | 后端运行环境 |
| MySQL | 8.0+ | 数据库 |
| Node.js | 18+ | 仅前端开发需要 |

---

## 🚀 快速部署

### 方式一：一键启动（推荐）

1. 确保已安装 **JDK 17+** 和 **MySQL 8.0**
2. 配置 MySQL 数据库连接（见下方数据库配置）
3. 双击运行 `一键启动.bat`
4. 等待构建完成后自动启动

### 方式二：手动启动

```bash
# 1. 构建项目
cd server
gradlew.bat :api:bootJar -Pdev

# 2. 启动服务
cd ..
java -jar server/api/build/libs/surveyking-v1.12.0.jar --spring.profiles.active=dev
```

---

## 🗄️ 数据库配置

### 1. 创建数据库

```sql
-- 登录 MySQL
mysql -u root -p

-- 创建数据库
CREATE DATABASE surveyking DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
EXIT;
```

### 2. 修改数据库连接

编辑 `application.properties` 文件：

```properties
# 数据库连接地址
spring.datasource.url=jdbc:mysql://localhost:3306/surveyking?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true

# 数据库用户名
spring.datasource.username=root

# 数据库密码（修改为你的密码）
spring.datasource.password=123456
```

## 🔧 常见问题

### Q1: 启动报错 "端口被占用"
```bash
# 查找占用端口的进程
netstat -ano | findstr :7007

# 结束进程（PID替换为实际值）
taskkill /PID <PID> /F
```

### Q2: 数据库连接失败
- 确认 MySQL 服务已启动
- 检查 `application.properties` 中的用户名密码是否正确
- 确认数据库 `surveyking` 已创建

### Q3: 忘记管理员密码
执行以下 SQL 重置密码：
```sql
UPDATE t_user SET password='$2a$10$xJG.zAFQvTvPRlMeuMe3H.qJZh0z0XxhJ9kUq.m3nTGBb7g7mMvHO' WHERE username='admin';
```
重置后密码为：`123456`
